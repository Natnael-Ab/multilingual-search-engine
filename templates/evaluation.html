Harmonic mean of precision and recall — balances both.

How Evaluation Works

Ground-truth relevance judgments were manually assigned for {{ tfidf_results | length }} test queries. Each query has a set of documents that a human annotator judged as relevant. The system runs each query with a given ranking method, retrieves up to 20 results, and computes the standard IR metrics above.

Cross-lingual queries (e.g. "health technology") include both English and Amharic documents in their ground truth, testing the system's ability to retrieve relevant documents across language boundaries.
